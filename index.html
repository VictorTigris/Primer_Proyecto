Primer día

Por fin hemos empezado.
Un comienzo un poco raro para ser la primera entrada de este cuaderno de vitácora, ¿no?. No lo sería tanto si supieran todo lo que hay detrás, pues resulta que no es la primera vez que intento empezar un proyecto como este, no. Ya he perdido la cuenta para serles sincero.
Pero el pasado pasado es. Esta vez parece ir todo mucho más encaminado que las otras veces.
Veamos qué hemos conseguido por el momento:

Un curso gratuito
|edTeam|

Lo más importante para empezar es una buena guía, y si es gratuita mejor que mejor.

Un buen editor de texto
|Visual Studio Code|

Un programa sencillo (aunque difícil de dominar) para comenzar a echar código.

Un generador de servidor
|Prepros|

Este tipo de programa nos facilitará la compresión de nuestro trabajo haciendo ver el resultado final de cada uno de nuestros cambios.

Ir al comienzo

Con todo el material para comenzar preparado, iniciamos sesión en edTeam y nos ponemos directamente con los cursos.
Lo recomendado, a la hora de aprender desarrollo web es conocer el tipo de estructura interna que vamos a manejar:

HTML
La estructura base de todo sitio web está escrita en HTML, que es un lenguaje de marcado.

Wiki te lo explica

CSS
Este tipo de archico,(.css) definirá el estilo o aspecto que tendrá nuestro sitio web. Es un paquete de estilo en cascada.

Wiki te lo explica

JavaScript
Este será el archivo que controlará las funciones internas del sitio. Este es el único lenguaje de programación propiamente dicho que vamos a manejar.
