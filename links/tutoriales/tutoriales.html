Inicio Horarios

Aquí encontrarás todos los tutoriales para descargar y preparar los programas necesarios para crear tu primer Blog.
Lo primero a tener en cuenta es que para crear cualquier tipo de sitio web necesitamos realizar desarrollo web. Por esta razón necesitamos, como base, manejar tres lenguajes:

  • Un lenguaje de marcado: HTML
  • Un lenguaje de hojas de estilo: CSS
  • Un lenguaje de programación para animar tu web: JavaScript

Es verdad que existen otros lenguajes para estas tres funciones, pero he optado por elegir los tres más conocidos. A fin de cuentas, cuanto más usado sea un lenguaje mayor será su comunidad.

Vamos paso a paso

Primer paso

Lo primero es lo primero, y no tiene nada que ver con echar código. Se trata de dominar la lógica.
Sí, lo sé, parece el tópico más común de los sitios web dedicados a la enseñanza de programación. Al principio yo tampoco lo entendía del todo, pero créeme, lo entenderás. Y más vale que te agarre con conocimiento que sin él, a no ser que quieras meterte en enormes quebraderos de cabeza.
Gracias a la lógica seremos capaces de planificar cómo resolver los problemas a los que vamos a enfrentarnos. Porque programar al final es eso, resolver problemas.
Así que asegúrate de dominar la lógica antes de empezar a echar código

Segundo paso

Ahora sí, ya podemos empezar a echar código como locos. Pero, ¿dónde? Esa es una muy pregunta.
Existen dos tipos de programas para escribir código de manera eficiente:


  • Editores de texto
  • IDEs (Entorno de Desarrollo Integrado)

Los editores de texto son más simples, ideales para usar al comienzo, ya que no vamos a necesitar ninguna herramienta compleja para escribir nuestros primeros códigos.
